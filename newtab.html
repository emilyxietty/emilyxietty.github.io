New Tab Chrome Extensions

JavaScript (+jQuery), HTML, CSS, Manifest v3

I've recently taken up the task to try making Google Chrome extensions. I was inspired to make my first extension, Ghiblify, because I wanted more personalization in my new tabs, which was not available in the current extension I was using. In addition, Studio Ghibli recently (in 2020/21), released 1,000 images from their movies, free for personal use. As an avid fan, I decided to use this as an opportunity to liven up my new tabs.
