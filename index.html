My Recent Projects

Debate Timer App


SwiftUI + JSON integration
This is my first iOS app, and my first time using SwiftUI. This app offers timers tailored to a variety of debate formats commonly used in the competitive circuit. This timer will ring, vibrate, and change colour in accordance to time signals that are typically given in debates. Additionally, I have added functions including information about the various debate formats, and a motion randomizer. I connected this application to hellomotions.com, a commonly-used website by competitive debaters to browse past tournament topics.

Debate Timer Site

HTML5/CSS3, JavaScript
This timer is a project that I made for the competitive debate circuit. I used this project to experiment with UX/UI design (specifically neumorphism). The timer changes color and rings when started/paused, when protected time is over/restarts (30 or 1 min), when grace time begins (last 15 secs), and when speaking time is over.

Duolingo Vocab Chrome Extensions


Manifest v3, HTML5/CSS3, JavaScript
Branched from Make America Kittens Again (MAKA), and adjusted + tailored for another purpose. These projects were my first Google Chrome Extensions. I made a Chinese (name: Chinglish) and Korean (name: Koreanish) version of this extension. I was inspired to make this because I have been learning Chinese and Korean on Duolingo for the past few months. These extensions swap out words on pages with their Chinese/Korean counterparts, and the swapped vocab can be adjusted by duolingo level.

This app is now available on the Chrome Web Store
